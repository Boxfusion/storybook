FAQ

Below are the most popular questions that we get asked about our platform.

Where do I get the library?

You can use npm to download the dependencies from the Boxfusion NPM server or download it.

How do I contribute?

You can go to the Github page and fork the source and make a pull request with the component or updates.

Where is the documentation?

Each component will have storybook or styleguidist documentation associated with it.

There are things missing!

Yes. Currently the Boxfusion Elements project is ongoing and comonents and documentation are added as needed.
